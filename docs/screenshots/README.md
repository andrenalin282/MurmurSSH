# Screenshots

This directory contains screenshots used in the project README and website.

## Naming convention

Use descriptive, lowercase, hyphen-separated filenames:

```
murmur-ssh-<screen-name>.png
```

Examples:
- `murmur-ssh-file-list.png` — SFTP file browser
- `murmur-ssh-connection-profile.png` — Profile creation/edit form
- `murmur-ssh-settings.png` — Settings dialog
- `murmur-ssh-passphrase.png` — SSH key passphrase prompt

## Format

- PNG preferred
- Capture at a representative window size (960×640 is the default)
- Light and dark theme variants may be included with a `-light` or `-dark` suffix
